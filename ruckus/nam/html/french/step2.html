Sur le premier écran s’affiche votre tableau de bord. Ce dernier vous permet d’avoir une vue d’ensemble de votre réseau :

lieux

points d’accès (PA)

clients

Comme vous pouvez le constater, il vous indique également si des alarmes sont activées. La notification d’alarme vous indiquera si un événement affectant votre réseau s’est produit. Par exemple, si un point d’accès est déconnecté.

Vous pouvez configurer tous les types de lieux — un emplacement géographique, un bâtiment ou une zone fonctionnelle de votre choix !
